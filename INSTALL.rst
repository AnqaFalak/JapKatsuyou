**INSTALL**

* **Ubuntu & MINT** : tested on Ubuntu 12.10 (quantal) and Mint 14 (Nadia):
    
 .. code-block:: sh

        $ sudo add-apt-repository ppa:kariminf/japkatuyou
        $ sudo apt-get update
        $ sudo apt-get install jpconj
