ConjFrame::doConj This function is used to show different conjugation forms.

This function search for the existance of the verb, using Edict2 module. Then, it calls for:

  • ConjFrame::basicConjugation To show the standard and basic conjugation forms.
  • ConjFrame::complexConjugation To show the complex conjugation forms.
