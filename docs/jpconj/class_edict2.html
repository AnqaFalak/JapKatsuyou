void Edict2::connect ( )

Edict2::connect Connect to edict2verbs.db SqLite database.

This function is loaded when we want to manage edict2verbs.db SqLite database. It looks for a connection name called "dbverbs", if found we just use the database assigned to that connection name. If not, we search for edict2verbs.db, and assign connection name called "dbverbs" to it.

void Edict2::desconnect ( )

Edict2::desconnect Desconnect edict2verbs.db SqLite database.

It looks for a connection name called "dbverbs", if found we remove it.
