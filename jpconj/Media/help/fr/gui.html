Interface utilisateur

Cette application offert une interface utilisateur facile à utiliser. Elle est conçu pour aider l'utilisateur à conjuguer les verbes japonais et faire d'autres choses de manière simple. L'application contient cinq parties:

  1. Barre de menus
  2. Barres d'outils
  3. Panneau de recherche
  4. Résultat de conjugaison
  5. Exemples
En outre, l'interface utilisateur est traduite en d'autres langues que le français: arabe, anglais, japonais.
Si vous souhaitez ajouter votre langue, vous pouvez nous aider en envoyant une demande à notre liste de diffusion du groupe d'utilisateurs: dzcoding@googlegroups.com
Ou, vous pouvez participer directement dans notre répertoire Github: https://github.com/AnqaFalak/JapKatsuyou

Panneau de recherche

Ici, vous pouvez entrer le verbe japonais que vous souhaitez conjuguer. Le verbe d'entrée doit atteindre certaines conditions:

  1. Il doit être en forme de dictionnaire (forme U). Par exemple: 行く, 食べる, 飲む, 遊ぶ, 信じる, 勉強する, etc.
  2. Il doit être en caractères japonais (Hiragana, Kanji). Le programme n'accepte pas les verbes écrits en caractères latins.
  3. Les noms qui acceptent le verbe auxiliaire する, doivent être suivies par lui afin qu'ils puissent être conjugués.
Ensuite, vous pouvez soit appuyer sur la touche "Entrer" ou le bouton "Conjuguer" pour obtenir le résultat.

Information sur le verbe

L'application vous donne des informations sur le verbe d'entrée. Si le verbe n'existe pas dans la base de données, elle vous affiche un message disant ça. Sinon, elle vous indique le type de verbes.
Ceci est un liste de la plupart des types importants de verbes:

  • Verbe Ichidan;
  • Verbe Godan - classe spéciale -aru;
  • Verbe Godan avec terminaison en 'bu';
  • Verbe Godan avec terminaison en 'gu';
  • Verbe Godan avec terminaison en 'ku';
  • Verbe Godan - classe spéciale Iku/Yuku;
  • Verbe Godan avec terminaison en 'mu';
  • Verbe Godan avec terminaison en 'nu';
  • Verbe Godan avec terminaison en 'ru';
  • Verbe Godan avec terminaison en 'su';
  • Verbe Godan avec terminaison en 'tsu';
  • Verbe Godan avec terminaison en 'u';
  • Nom ou participe prenant le verbe auxilaire Suru;
  • ... etc.

Conjugaison standard

Elle est utilisée par les Japonais pour apprendre la conjugaison des verbes.

Conjugaison de base

Elle montre les radicaux de base: a e u i o plus la forme t.

Conjugaison complexe

Elle montre certaines formes complexes, qui sont originaires des radicaux de base suivis par des suffixes.
