JapKatsuyou

JapKatsuyou est une application open source et gratuite, utilisée pour la conjugaison des verbes Japonais.

  • Elle est destinée pour les apprenants de langue japonaise, dans l'espoir qu'elle permettra de simplifier la tâche de conjugaison.
  • Aussi, elle peut être utilisée pour simplifier la composition des phrases pour les apprenants avancés, étant donné qu'elle contient des formes complexes.

Fonctionnalités de JapKatsuyou

L'application peut fournir les fonctionnalités suivantes:

  • Chercher la présence d'un verbe, en utilisant sa forme dictionnaire. Ex. 食べる, 飲む, 行く, etc.
  • Si le verbe est trouvé, afficher son type. Ex. 食べる: verbe ichidan.
  • La base de données des verbes contient les noms qui sont utilisés avec le verbe auxiliaire "する". Ex. 勉強する, 選択する, 連絡する, etc.
  • L'interface utilisateur est traduite en quatre langues: arabe, anglais, français et japonais.
  • Les tableaux de conjugaison peuvent être exportés vers trois formats: pdf, odt, and htm.
